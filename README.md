# AtmosFX — 3D Visual Artist Portfolio

A premium, dark-themed portfolio website for a 3D visual artist. Built entirely with vanilla HTML, CSS & JavaScript — zero dependencies, zero frameworks.

## ✨ Features

- **Procedural Canvas Artwork** — Each portfolio item is rendered in real-time using the Canvas API with custom shaders, gradients, and organic shapes
- **Embedded Video Showreel** — YouTube video plays directly on-site with custom play/pause and fullscreen controls
- **Scroll Reveal Animations** — Intersection Observer-powered fade-in transitions throughout the page
- **Cinematic Design** — Dark aesthetic with neon accent lighting, grain overlays, and glassmorphism effects
- **Fully Responsive** — Adapts seamlessly from desktop to mobile
- **Contact Form** — Integrated inquiry form with project type and budget selectors

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Structure | HTML5 Semantic |
| Styling | Vanilla CSS (custom properties, grid, flexbox) |
| Logic | Vanilla JavaScript (Canvas API, Intersection Observer) |
| Fonts | Google Fonts — Syne, DM Mono, Cormorant Garamond |
| Video | YouTube IFrame Embed API |

## 🚀 Getting Started

Clone and open locally:

```bash
git clone https://github.com/Venom4522/AtmosFX.git
cd AtmosFX
```

For full functionality (including YouTube embed), serve via HTTP:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`
